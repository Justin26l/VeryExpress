# VeryExpress
an generator that generate [Express](https://github.com/expressjs/express) rest-api app based on [jsonSchema](https://github.com/json-schema-org) and [openApi](https://github.com/OAI) 

## tech stack
- TypeScript
- ExpressJs
- mongoDb
